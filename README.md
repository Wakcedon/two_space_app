# TwoSpace

Мессенджер на основе *Flutter*, созданный для проектной деятельности в колледже. 

Функционал мессенджера будет представлять собой объединение функций Telegram, Discord и Widgetable.

На данный момент TwoSpace (Не форкнутый) находится в стадии альфа-разработки в закрытом тестировании, наш клиент добавит ещё Matrix, Telergam, Signal, irc, хз нахуя 

### Технологии
1. Dart - язык программирования от Google, основа функций и экранов
2. Flutter - комплект средств разработки для создания приложений на Android, Windows, Linux, MacOs и iOS
3. AppWrite v1.8 - основа Backend разработки и серверной части проекта
4. Node.js - язык программирования для скриптов и серверных функций
