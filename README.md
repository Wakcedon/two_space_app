# TwoSpace

Мессенджер на основе *Flutter*, созданный для проекта в колледже.
Функционал мессенджера будет представлять собой объединение функций Telegram, Discord и Widgetable в современном исполнении.
На данный момент TwoSpace находится в стадии альфа-разработки в закрытом тестировании.

## Технологии
1. Dart SDK v3.9.2 - это набор инструментов и библиотек, которые разработчики используют для создания приложений на языке программирования Dart, созданным Google, основа функций и экранов приложения
2. Flutter v3.35.7 - комплект средств разработки для создания приложений на Android, Windows, Linux, MacOs и iOS
3. AppWrite v1.8.0 - платформа с открытым исходным кодом (open-source) для создания серверной части (backend) приложений, основа разработки
4. Node.js v24.11.0 - платформа с открытым исходным кодом для работы с языком JavaScript, построенная на движке V8 от Google, среда выполнения JavaScript для скриптов и серверных функций
5. DevTools 2.48.0 - это комплексный веб-инструментарий компании Google для Dart- и Flutter-приложений, предназначенный для интерактивной отладки, профилирования, анализа производительности и оптимизации кода
